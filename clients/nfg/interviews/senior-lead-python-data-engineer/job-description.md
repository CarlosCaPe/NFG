# Senior/Lead Python Data Engineer

**Source**: https://www.newfireglobal.com/careers/senior-lead-python-data-engineer/
**Region**: LATAM (Costa Rica, Argentina, Colombia, Mexico)
**Type**: Fully remote
**Recruiter**: Leon Dueñas (guillermo.duenas@newfireglobal.com)

## The Project

U.S. healthcare organization operating in care management and quality improvement.
Modernizing data stack by consolidating two existing overlapping data solutions that rely on legacy SQL- and SSIS-based pipelines.
Transitioning to a modern, scalable, Python-based data platform — improving performance, maintainability, and long-term sustainability.
Regulated environment (healthcare, HIPAA/HITRUST).

## Requirements

- 8+ years data engineering or backend platform roles, deep hands-on Python
- Strong experience with Python, Dagster, Snowflake, and/or Databricks
- Proven ability to define and enforce engineering standards, patterns, guardrails
- Experience leading technical decision-making on high-impact architectural changes
- Strong CI/CD, dependency management, testing, release practices for Python
- Hands-on GitHub and GitHub Actions for CI/CD
- Solid data modeling, orchestration, workflow design
- Experience in regulated environments, preferably healthcare
- HIPAA / HITRUST awareness for sensitive data
- Strong troubleshooting + ownership mindset for production systems
- SSIS experience is a strong plus

## Day-to-Day

- Design, build, maintain production Python data pipelines (Dagster, Snowflake, Databricks)
- Define, evolve, document Python engineering standards and guardrails
- Lead adoption of version-locked dependency management and standardized workflows
- Build, maintain, improve GitHub Actions CI/CD pipelines
- Review complex designs and code for correctness, security, performance
- Ensure reliable pipelines with error handling, retries, idempotency, validation
- Anticipate failure modes — design for recoverability and operational clarity
- Troubleshoot CI/CD failures, dependency conflicts, production incidents (RCA)
- Apply secure development practices for healthcare data (secrets management, least privilege)
- Collaborate with platform architecture and data platform teams
- Mentor senior engineers through design reviews and knowledge sharing
- Participate in hiring processes / interview panels
